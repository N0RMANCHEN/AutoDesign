# Plan Title

## Summary

一句话说明这项 active work 要解决什么问题。

## Scope

- 列出本次计划覆盖的改动面

## Dependencies

- 相关模块、文档、脚本或外部前置条件

## Entry Conditions

- 开始实施前必须满足的条件

## Workstreams

- 拆出 2-4 条主要实施工作流

## Closure Tasks

- 列出离 task 关闭还差哪些明确子任务
- 不写 `done / in_progress`
- 默认至少 3 条

## Exit Conditions

- 什么结果算完成

## Risks

- 主要风险和失败模式

## Rollback

- 回滚策略

## Verification

- 准备执行的验证方式和最低门槛
